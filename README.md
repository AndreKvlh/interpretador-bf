# Interpretador de Código Brainfvck
Este é um interpretador desenvolvido em Java voltado a processar a linguagem de programação esotérica (EsoLang) mais conhecida de todas: a **Brainfvck**.

O projeto foi construído focando na manipulação direta de memória e operações de I/O, simulando o comportamento de uma máquina de Turing com uma fita de células e um ponteiro de dados. É uma excelente ferramenta para entender como o processamento de baixo nível funciona através de uma interface de alto nível.

## 🚀 Modo de usar
Para facilitar a experiência do usuário, o projeto conta com um instalador que configura automaticamente o sistema para uso via linha de comando (CLI).

### 1. Instalação
- Vá até a aba Releases deste repositório.

- Baixe o arquivo executável mais recente (ex: Setup_InterpretadorBF.exe).

- Execute o instalador em seu computador. Ele irá copiar os arquivos necessários e configurar a variável de ambiente Path automaticamente.

### 2. Execução
Após instalado, você pode executar seus scripts de qualquer lugar do sistema sem precisar configurar caminhos manualmente:

- Abra o seu Prompt de Comando (CMD) ou PowerShell.

- Navegue até a pasta onde contém seu arquivo .b ou .bf.

- Digite o comando seguido do nome do seu arquivo ```interpretadorbf [nome-do-arquivo].bf```

## 💻 Tecnologias
- Java 23: Versão utilizada para o desenvolvimento e compilação.

- Maven: Responsável pelo gerenciamento de build e ciclo de vida do projeto.

- Inno Setup: Utilizado para gerar o instalador e automatizar a configuração do Path no Windows.
